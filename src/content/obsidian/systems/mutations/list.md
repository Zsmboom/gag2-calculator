## 🧮 Value Formula

```
Crop Value = BaseFormula(weight) × Mutation_Mult × (1 + FriendBoost%) × Quantity
```

Where `BaseFormula(weight)` is a per-crop fitted curve:
- **Power fit:** `coeff × weight^exp` (exp ranges **2.0–3.4** per crop)
- **Quadratic fit:** `a·w² + b·w + c` (for some crops)

> **Mutations are MUTUALLY EXCLUSIVE.** Only one applies at a time. This is the biggest change from GAG1.
