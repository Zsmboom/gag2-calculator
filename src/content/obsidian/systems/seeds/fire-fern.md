# 🌿 Fire Fern — GAG2 Legendary Crop

Fire Fern is a **Legendary multi-harvest crop** in Grow a Garden 2, costing **6,000,000 Sheckles** or **659 Robux** per seed with a restock chance in the seed shop. Added in **Update 1.13.0**.

## Key Stats

| Attribute | Value |
|-----------|-------|
| Rarity | LEGENDARY |
| Seed Price | 6,000,000 🪙 / 659 💎 |
| Average Value | 900 Sheckles (1kg) |
| Average Weight | 9 kg |
| Harvest Type | Multi |
| Obtainable | Yes |
| Seed Shop | Listed |

## Overview

Fire Fern is a Legendary multi-harvest crop added in Update 1.13.0 (early July 2026). With a 6M Sheckle seed cost, it sits between Cherry (1.2M) and Sunflower (5M) in price range, offering solid multi-harvest returns. Its base value of 900 S at 1kg makes it comparable to Mythic-tier crops like Pomegranate.

## How to Get Fire Fern

- Buy from the Seed Shop for 6,000,000 Sheckles or 659 Robux
- Listed at the seed shop (standard rotation)

## How much is Fire Fern worth?

The value is calculated using the standard GAG2 formula:
```
Sell Price = Base_Value(1kg) × (Harvest_Weight / Avg_Weight)² × Mutation_Mult × (1 + FriendBoost%) × Quantity
```

At average weight (9kg): 900 × (9/9)² = 900 Sheckles per unit
At max weight (100kg): 900 × (100/9)² ≈ 111,111 Sheckles per unit
